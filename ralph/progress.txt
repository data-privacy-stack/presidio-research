## Codebase Patterns
- `InputSample.tokens` is a spaCy Doc or list; use `str(token)` to get token text in either case
- `InputSample.sample_id` may be None — fall back to loop index for `sentence_id`
- ruff is the linter: `ruff check <file>` — fix unused imports before committing
- Test conftest.py requires spacy/pandas/presidio_analyzer; regular tests run fine with it
- `MockTokensModel` in tests/mocks returns a fixed prediction list — useful for DataFrame schema tests
- **Evaluator no longer accepts entity_mapping or compare_by_io** — stripped in US-005; evaluator expects pre-mapped IO-format data
- **PresidioRecognizerWrapper default labeling_scheme is now "IO"** (changed from "BILUO" in US-005)
- **SpacyModel tests deleted** in US-005 (SpacyModel itself will be removed in US-010)
- test_entity_hierarchy.py and tests/integration/test_notebook.py are the legitimate excludable/tricky tests; run with `--ignore=tests/test_entity_hierarchy.py` but integration now passes
